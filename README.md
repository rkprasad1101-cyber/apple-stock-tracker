# 🍎 Apple Store Pickup Tracker

A Python-based tracker that monitors Apple India Store Pickup availability for selected iPhone models and sends instant Telegram notifications.

## Features

- ✅ Store Pickup Monitoring
- ✅ Telegram Notifications
- ✅ GitHub Actions Automation
- ✅ No Duplicate Alerts
- ✅ Supports Multiple Colors
- ✅ Easy Configuration
- ✅ Runs Every 10 Minutes

## Current Configuration

- Country: India
- Store: Apple Saket (Select Citywalk)
- Product: iPhone 17
- Storage: 256GB
- Colors: All

## Technologies

- Python 3.11
- GitHub Actions
- Telegram Bot API
- Requests
